# resumeranker Chatbot

This repository contains a minimal example of a Streamlit application that wraps a `chatterbot` chatbot. The bot is trained on the default English corpus and can be run locally with Streamlit.

## Running the app

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Start the Streamlit server:

```bash
streamlit run app.py
```

Then open the provided URL in your browser to chat with the bot.
